06 — Usage
/* ── Text node: formats on load ─────────────── */ <span is-pattern="(000) 000-0000">3361234567</span> /* ── Input: real-time as user types ─────────── */ <input is-pattern="(000) 000-0000"> /* ── Manual API ──────────────────────────────── */ window.IsPattern.format('3361234567', '(000) 000-0000') // → "(336) 123-4567" /* ── Pattern tokens ──────────────────────────── */ 0 → digit (0–9) A → letter (a–z A–Z) * → any character $ → currency prefix (triggers currency mode) everything else → literal (kept as-is)
